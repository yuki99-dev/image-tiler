# JP - Python Image Tiler（Tk GUI）
**カスタマイズ可能なタイルサイズ、画像全体およびタイルごとの最大色数、画像リサイズ、プレビュー、.png形式へのエクスポート機能を備えた、Tkinterベースのタイル画像生成ツールです。**
組み込みシステム、レトロスタイルのグラフィック制作、アセットの素早い準備に最適です。

## 特徴
- **Tkinterで構築されたGUI**
- **カスタマイズ可能なタイルサイズおよび画像サイズ**
- **色数制限機能（画像全体およびタイルごと）**
- **タイル出力のライブプレビュー**
- **組み込み／ファームウェア用途向けに.png形式でエクスポート**

## 使い方
スクリプトを実行し、GUIから画像を読み込みます。
タイル設定や色数制限を調整し、準備ができたらエクスポートしてください。

# EN - Python Image Tiler (Tk GUI)
**A Tkinter-based tile image generation tool with customizable tile sizes, maximum number of colors for the entire image and per tile, image resize, preview, and export to .png format.**
Ideal for embedded systems, retro-style graphic production and quick asset preparation.

## Features
- **GUI built with Tkinter**
- **Customizable tile and image sizes**
- **Color limit function (whole image and per tile)**
- **Live preview of tile output**
- **Export in .png format for embedded/firmware use**

## How to use
Run the script and load the image from the GUI.
Adjust tile settings and color limits and export when ready.
